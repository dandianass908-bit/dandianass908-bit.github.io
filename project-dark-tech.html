Dark Tech – Inclusive Robot Project

Een innovatief project waarin we onderzochten hoe sociale robots zowel een dark side als een positive side kunnen hebben. Van ethische risico’s tot inclusieve culturele interacties met een echte NAO robot.

Project Overzicht

Dit project draaide om het onderzoeken van de impact van sociale robots op mensen. Eerst onderzochten we de dark side: manipulatie, afhankelijkheid en emotionele beïnvloeding. Daarna werkten we aan een positieve toepassing waarin robots mensen juist kunnen verbinden.

Voor het Diemen 800 project ontwikkelden we een concept waarbij een robot verschillende culturen vertegenwoordigt via dans, interactie en inclusieve communicatie.

Mijn rol: onderzoek, conceptontwikkeling, prototyping, werken met Choregraphe software en het ontwerpen van robot interacties.
